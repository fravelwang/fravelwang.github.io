序号 命令 说明

十九、高危命令注意

  • jump / set $rip → 改变执行流，可能破坏堆栈
  • call → 调用目标代码可有副作用
  • set var → 修改关键逻辑需谨慎
  • detach → 会恢复执行，请确保安全
  • signal → 可能导致即时终止或异常

二十、调试心法小结

  1. 所有命令皆为工具，不为表演 —— 优先录制自动化。
  2. 先追条件，再断行为 —— 用 break if 不要撒网。
  3. 多线程永远采集全景 —— 一线程异常多线程有因。
  4. 善用脚本记录可重复性。
  5. core 是真相，不依赖复现。
  6. 源码 + 汇编 = 真图谱。
  7. 越自动越低风险。
  8. 永不在生产 attach 长时。
  9. 多用 batch 模式采样。
  10. 离线分析，安全优先。
