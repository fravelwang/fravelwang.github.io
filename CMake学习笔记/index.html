CMake学习笔记

开篇先放两个链接, 一个是教程, 一个是小抄

CMake的安装就不赘述, 官网下最新版本即可

CMake Cookbook小抄

源码构建工具集:

  • cmake 描述 项目的 可执行文件/库 在各种操作系统上的 配置 构建 和 安装
  • ctest 定义测试 测试套件 及 如何执行
  • cpack 打包专用DSL(domain-specific language)
  • cdash 面板中 展示测试结果的工具

书的内容: cmake系列基本使用, 模块化, 打包, 重构cmake工程文件

cmake 的 时序图

构建目录(build目录)中

  • cmake time 或 configure time: cmake程序的运行时, 此过程处理CMakeLists.txt并配置
  • generation time: cmake将生成本地原生构建工具(msbuild/make…)构建脚本
  • build time: cmake调用原生构建工具, 并执行脚本调用原生编译器进行编译
  • ctest time 或 test time: 构建编译完成, 运行测试套件, 进行结果验证
  • cdash time 或 report time: 将测试结果上传至面板, 供查看

构建目录外(一般项目目录)

  • install time: 将项目的 可执行文件, 库, 源文件, obj文件 从 构建目录 安装到 指定目录, 此时已经可以直接运行或作为库引入其他项目

全局安装

  • cpack time 或 packaging time: 打包 程序和源码
  • package install time: 系统范围内安装生成的包
