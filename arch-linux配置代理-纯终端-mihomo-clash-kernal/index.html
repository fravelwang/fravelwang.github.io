默认wget在没有代理时从github下载东西经常失败，所以第一步安装axel下载器

我的是arch linux,用pacman

下载代理工具mihomo解压安装到自定义位置，我选择/usr/local/bin目录
地址https://github.com/MetaCubeX/mihomo/releases

解压

得到
gunzip mihomo-linux-amd64-v1-v1.19.23可执行文件

添加执行权限

移动文件到/usr/local/bin

创建配置文件

编辑配置, 我用vim, 不习惯的可以用nano

如下是配置内容

配置额外说明：
mixed-port: 7890 → 本地代理端口，HTTP/HTTPS/SOCKS 共用。
external-controller: 127.0.0.1:9090 → 控制接口端口，可以用 Clash Dashboard 管理。
proxy-providers → 引用你的订阅地址，自动拉取节点。
proxy-groups → 定义一个选择组，把订阅里的节点放进去。
rules → 简单规则：国内直连，其他走代理。

手动下载Country.mmdb文件, geoip.metadb文件比较小会自动下载
github地址

shell添加环境变量，我的是bash

INFO日志出现match using proxy[compatible]算配置成功,开始转发代理数据

切后台执行
切后台命令
按组合键ctrl + z
然后
